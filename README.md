# notes-mdl

Personal study notes and templates for Machine Learning courses.

## Contents

- `tmp.typ` - Main notes template
- `sheet-v2.typ` - Sheet template module

## Usage

These are personal learning materials created using [Typst](https://typst.app/).

```bash
typst compile tmp.typ
```
