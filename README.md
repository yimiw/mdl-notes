# A. notes-mdl

Personal study notes and templates for Machine Learning courses.

## 1. Contents

- `tmp.typ` - Reusable template module (shared across courses)
- `25HS_PAI_sheet-v2.typ` - Probabilistic AI course cheat sheet


## 2. Usage

Notes are created via [Typst](https://typst.app/).

```zsh
typst compile tmp.typ
```
